# Tienda LUMARK 

Este es el repositorio de **Tienda LUMARK**, mi proyecto (pre-entrega) del curso de **Front-end JS** que estoy cursando.

---

## ¿Qué es Tienda LUMARK?

Tienda LUMARK es una **tienda online de accesorios deportivas**.

Básicamente, es una página web de comercio electrónico (e-commerce) simple para:

1.  Mostrar un catálogo de **productos destacados** (las camisetas).
2.  Implementar un **header responsive** con buscador y navegación.
3.  Incluir secciones interactivas como **reseñas** y un formulario de **contacto**.
4.  Asegurar que todo se vea bien tanto en el escritorio como en un celular.

---

## ¿Por qué es útil este proyecto?

Este proyecto es super útil como **demostración de las habilidades que pude aprender hasta ahora**. Muestra que puedo:

* **Estructurar HTML** de manera semántica.
* Aplicar **diseño con CSS**, incluyendo Flexbox y Grid para crear layouts dinámicos.
* Aplicar **adaptabilidad** usando `@media queries` para que la página funcione en pantallas de 600px o menos (como un celular).
* Incluir **navegación y formularios**.

---

## ¿Cómo Empezar? 

Para ver la página o seguir trabajando en ella, solo tenés que seguir estos pasos:

1.  **Clonar el repositorio:**
    ```
    git clone [https://github.com/tuusuario/repoLucaNeumark.git](https://github.com/tuusuario/repoLucaNeumark.git)
    ```
    *(Asegúrate de cambiar la URL por la de tu repositorio)*

2.  **Abrir el proyecto:**
    Navegá hasta la carpeta clonada (`PROYECTO TT (PRE-ENTREGA)`) y hacé doble clic en el archivo **`index.html`** para abrir la página.

3.  **Explorar el código:**
    Abrí la carpeta en un editor de código (como VS Code) para ver la estructura.
